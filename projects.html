Research Project

Overview

LASA is a research-based intelligent railway seat allocation system that improves traditional booking by balancing passenger load across coaches using algorithmic logic.

Problem It Solves

Traditional railway systems allocate seats sequentially without considering coach load, leading to imbalance and inefficient seat utilization.

How It Works

Passenger details are entered.
System calculates coach-wise load.
Least-loaded coach is selected.
Seat assigned based on preference + balance.
Seat chart updates dynamically.
Freed seats are reused mid-journey.

Tech Stack

Python Algorithm Design Data Simulation Visualization

Challenges & Learnings

  • Designing a fair load-balancing algorithm
  • Handling multiple passenger constraints
  • Understanding real-world optimization logic
